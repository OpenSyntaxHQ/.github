# Code of Conduct

## Our Commitment

OpenSyntaxHQ is committed to providing a welcoming, respectful environment for everyone. We expect all participants to uphold these standards.

## Expected Behavior

- **Be respectful** — Treat others as you'd want to be treated
- **Be constructive** — Offer solutions, not just criticism
- **Be professional** — Keep discussions focused and productive
- **Be patient** — Not everyone has the same experience level
- **Be accountable** — Own your mistakes and learn from them

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, inflammatory comments, or bad-faith arguments
- Publishing others' private information
- Dismissive or condescending responses to questions
- Any conduct that creates a hostile environment

## Scope

This code applies to all project spaces including:

- GitHub repositories, issues, and pull requests
- Project discussions and comments
- Any official communication channels

## Enforcement

Violations may result in:

1. Warning with explanation
2. Temporary restriction from participation
3. Permanent ban from the organization

Maintainers will review reports confidentially and respond appropriately.

## Reporting

Report issues to the maintainers via:

- Email: conduct@opensyntaxhq.dev
- Private message to organization admins

All reports will be reviewed and handled with discretion.
